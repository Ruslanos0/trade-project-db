# Internet Library 
# Система управления интернет-библиотекой

## Обзор проекта
Полнофункциональная система управления интернет-библиотекой с учётом книг, категорий (жанров), прайс-листов и истории выдачи/покупок. Проект разработан в рамках курса по дисциплине "Базы данных".

## Основные возможности
- [cite_start]**Учёт книжного фонда** — полный CRUD-функционал для управления списком книг.
- [cite_start]**Постраничный вывод (Pagination)** — эффективная загрузка больших списков через эндпоинт `/api/book/paged`.
- [cite_start]**Категоризация** — иерархическая структура групп книг (жанров).
- [cite_start]**Документация Swagger** — интерактивное тестирование всех API-запросов прямо из браузера.
- [cite_start]**Docker-контейнеризация** — готовое окружение для быстрой развёртки PostgreSQL и Node.js приложения.

## Схема базы данных
### **Основные таблицы:**
1. [cite_start]**`books_group`** — Категории и жанры книг.
2. [cite_start]**`books`** — Данные о книгах (название, описание, связь с группой).
3. [cite_start]**`price_list`** — Прайс-листы для платных услуг библиотеки.
4. [cite_start]**`purchase`** — Транзакции выдачи или продажи книг.

### **Связи:**
- **Один-ко-многим**: Категория → Книги.
- **Многие-ко-многим**: Книги ↔ Прайс-листы.
- **Один-ко-многим**: Книга ↔ История операций.

## Технологический стек
- **Backend:** Node.js, Express.js 
- **ORM:** Sequelize 
- **База данных:** PostgreSQL 
- **Контейнеризация:** Docker, Docker Compose 
- **Документация API:** Swagger/OpenAPI 3.0 
- **Тестирование:** Postman
